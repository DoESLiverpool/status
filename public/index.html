Status information available via our JSON API: /api/status

Each service has a state. You can see what each state number means by looking at the golang code below:

const (
    // UnkownState is for when we can not determine whether the service is working
    UnkownState ServiceState = 0

    // BrokenState is for when we know a service is broken
    BrokenState ServiceState = 1

    // WorkingState is for when we know a service is working
    WorkingState ServiceState = 2
)
